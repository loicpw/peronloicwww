install and test
=======================

dev install
****************

There is a makefile in the project root directory:
    
.. code-block:: bash

    $ make dev

Using **npm**, the above is equivalent to:

.. code-block:: bash

    $ npm install

run the tests
******************

Use the makefile in the project root directory:

.. code-block:: bash

    $ make test

This runs the tests generating a coverage report
